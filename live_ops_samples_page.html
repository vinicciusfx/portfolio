Live Ops Samples

Confira no YouTube

LiveOps Samples é um projeto pessoal feito com o objetivo de aplicar os recursos de Live Operations na prática dentro da Unity. Nele procurei aprender e aplicar todos os recursos disponíveis do Unity Services em um único projeto. É projeto essencialmente técnico, focado na programação dos sistema, porém fiz o máximo que pude para adicionar recursos visuais para facilitar a visualização das mecânicas.

Features

  • Persistência de dados do jogador via Cloud Save, garantindo armazenamento seguro, sincronização entre sessões e integridade do progresso.
  • Sistema de recompensas diárias distribuídas em um ciclo de 7 dias, com definição remota de conteúdo via Remote Config, permitindo ajustes sem necessidade de atualização do aplicativo.
  • Sistema de missões diárias, com progresso e distribuição de recompensas configuráveis via Remote Config.
  • Implementação de loja online atualizável em tempo real, com assets inseridos remotamente via Addressables e Unity Cloud Content.
  • Integração de sistema de economia integrado ao Unity Economy, gerenciando as moedas de forma centralizada.

Minhas resposabilidades

A ideia desse projeto nasceu de um atraso de semanas que minha inexperiência causou em um projeto que trabalhava. Foi planejado fazer uma loja com itens rotativos e que se atualizasse e alimentasse de novos itens sem a necessidade de atualizar o app em produção. Minha tentativa de programar isso falhou e com isso tivemos que reduzir o escopo dessa loja para prosseguir com o projeto. Dali em diante me dediquei ao estudo do assunto e construí o LiveOps Samples, um laboratório onde posso adicionar novas mecânicas, eventos e ajustes que podem ser manipulados via backend. Utilizei os serviços do Unity Game Services para todas as mecânicas presentes nesse projeto.
